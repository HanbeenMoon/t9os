# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities through [GitHub Security Advisories](https://github.com/HanbeenMoon/t9os/security/advisories/new).

Do **not** create public issues for security vulnerabilities.

## Scope

T9 OS is a personal operating system. Security concerns include:
- Exposed API tokens or credentials in code
- Unsafe file operations (especially in hooks)
- Injection vulnerabilities in seed engine queries
