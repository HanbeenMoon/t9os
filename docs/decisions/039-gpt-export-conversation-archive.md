# ADR-039: GPT 대화 기록 전량 아카이빙 — 1,070건 JSON 인덱싱

- 날짜: 2026-02-25
- 상태: 채택됨
- 결정: GPT export(conversations-000~010.json, 1,070건 대화)를 T9OS 데이터 아카이브로 이관하고, 구조를 분석하여 검색 가능하게 만든다. Claude AI 채팅 기록(83파일, 148,853건)도 동일하게 아카이빙한다.
- 이유: AI 채팅 기록은 설계자의 과거 사고 과정과 프로젝트 의사결정의 원천. T9 OS가 "모른다" 판단 전에 반드시 참조해야 할 데이터.
- 대안: GPT 웹에서 필요 시 검색 (오프라인 불가, 전문 검색 불가), 선별 아카이빙 (누락 위험)
- 결과: _legacy/_personal_dump/ 심볼릭 링크, CLAUDE.md 데이터 접근 경로 섹션에 명시, grep 기반 검색 체계 확립
- 출처: 20260225_CC_002_(2-t9)_015046_GPT_export_구조파악.txt

## Simondon Mapping
전개체적 장(preindividual field)의 보존: AI 대화라는 과포화된 잠재성을 삭제하지 않고 구조화하여 미래 개체화의 자원으로 유지.
