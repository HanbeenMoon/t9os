# ADR-062: research-project 트로이목마 전략 — 학술지급 + 학부제출용 이중 산출물

- 날짜: 2026-03-17
- 상태: 채택됨
- 결정: research-project 논문을 학술지급 정본(v25)과 학부제출용(트로이목마 v2) 두 버전으로 동시 생성한다. 학부제출용은 교수(advisor-A) 피드백을 반영한 축약본이며, 학술지급은 독자적으로 투고 준비한다.
- 이유: 학부 연구보고서 의무와 학술지 투고 목표를 동시 달성. 트로이목마 전략으로 학부 의무를 최소 비용으로 충족하면서 학술지급 품질은 유지.
- 대안: 단일 버전 (학부 의무와 학술지 기준 충돌), 학부용만 (학술지 투고 기회 상실)
- 결과: research-project_트로이목마_v2.docx+pdf, 연구설계_v1.md, 메타연구설계_v1.md 생성
- 출처: 20260317_CC_002_032100_research-project_v26_트로이목마하향.txt, 20260317_CC_004_220000_research-project_트목v2_pipeline-project_전체완료.txt

## Simondon Mapping
이접(disparation)의 창조적 해소: 학부 의무(축약)와 학술지 목표(완전) 사이의 양립 불가능한 긴장이 이중 산출물이라는 새 차원에서 해소됨.
