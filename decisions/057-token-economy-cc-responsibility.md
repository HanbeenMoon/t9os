# ADR-057: Token Economy — cc Tokens as Lifeblood

- Date: 2026-03-16
- Status: Accepted
- Decision: Minimize cc (Claude Code) token consumption. Delegate long scripts/code generation to cx (Codex). cc is reserved for judgment, routing, and report intake. "Tokens = lifespan." If a simple task takes over 2 hours, abort and move to the next stage.
- Rationale: cc tokens are expensive and rate-limited. Spending them on mechanical work (code generation, bulk processing) is waste.
- Outcome: Agent role allocation in L1

## Simondon Mapping
Resource allocation mirrors concretization — each agent converges on its optimal function rather than all doing the same thing.
