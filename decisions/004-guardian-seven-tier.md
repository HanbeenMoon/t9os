# ADR-004: Seven-Tier Guardian System (G1-G7)

- Date: 2026-03-16
- Status: Accepted
- Decision: Implement 7 specialized guardians for automated quality review:
  - G1 Tech: security, code quality, Build vs Buy
  - G2 Philosophy: G2-A (vision distortion prevention) + G2-B (ontological compliance), two-phase verification
  - G3 Rules: L1/L2 compliance, data rules, SRBB order
  - G4 Writing: external document quality
  - G5 Business: revenue model, cash flow viability
  - G6 Marketing: user acquisition, comprehension
  - G7 Design: visual quality, philosophical visualization
- Rationale: AI assistants quietly flatten thinking over repeated interactions. Without systematic review, vision dilution and rule drift are inevitable.
- Outcome: `constitution/GUARDIANS.md`, `pipes/gm_batch.py guardian`

## Simondon Mapping
The guardian system maintains individuation integrity — preventing the system from collapsing into conventional patterns that eliminate productive tension.
