# ADR-049: Syncthing + Tailscale Multi-PC Sync

- Date: 2026-02-20
- Status: Accepted
- Decision: Bidirectional workspace sync between two PCs via Syncthing over Tailscale VPN.
- Rationale: Work happens across multiple machines. File availability must be seamless.
- Outcome: Syncthing folder sync, Tailscale mesh network
