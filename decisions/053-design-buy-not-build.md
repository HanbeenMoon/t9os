# ADR-053: Design Is Buy, Not Build

- Date: 2026-03-17
- Status: Accepted
- Decision: Design work uses external tools (v0.dev, etc.) — Buy, not Build. The control tower does not generate design artifacts directly.
- Rationale: SRBB principle applied to design. Design quality requires specialized tools that exceed what a general-purpose AI agent can produce.
- Outcome: Design workflow via external tools, quality verified by G7

## Simondon Mapping
SRBB is a concretization principle — each tool should do what it does best rather than one tool attempting everything abstractly.
