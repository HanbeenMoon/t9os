# ADR-040: Legacy Data Digest Pipeline

- Date: 2026-03-15
- Status: Accepted
- Decision: Compress personal data archives (notes, messages, databases) into multi-layer digests: fact digests, thought maps, raw insights. Define access order: thought map → facts → insights → grep → originals.
- Rationale: 20,000+ legacy items cannot be searched efficiently without structured digests. The pipeline transforms unstructured archives into searchable knowledge layers.
- Outcome: Digest files, layered access pattern

## Simondon Mapping
Legacy data is sedimented preindividuality — not dead, but requiring a different access pattern to reactivate its potential.
