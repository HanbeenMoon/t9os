# ADR-032: ODNAR 비전 복원 — 메모앱에서 개인 온톨로지 인프라로 피봇

- 날짜: 2026-03-16
- 상태: 채택됨
- 결정: ODNAR의 핵심 정체성을 "메모앱 + 반대기록탐지"에서 "개인 온톨로지 인프라"로 재정의한다. AI가 왜곡한 표현(소름 돋는 순간, 메모앱, 반대 기록 탐지)을 전면 제거하고, 한빈 원문 기반의 PHILOSOPHY_ANCHOR.md를 생성한다.
- 이유: 과거 AI 세션들이 한빈의 원래 비전을 "메모앱 + 반대기록탐지"로 축소/왜곡함. 원문(카톡 대화, GPT 2025-11-18 "개인 온톨로지 아이디어")을 전수 탐색하여 본래 의도를 복원.
- 대안: 기존 "메모앱" 프레이밍 유지 (한빈 비전 왜곡), PKM 도구로 포지셔닝 (시장 포화)
- 결과: ontology-engine/ 디렉토리 생성, SQL 스키마 6테이블, KG 자동 추출 파이프라인, Divergence Score 실증, D3.js 시각화. 감시단 CATASTROPHIC → CLEAN.
- 출처: 20260316_CC_004_124500_ODNAR_CTO_세션.txt, 20260317_CC_003_032503_야간자율작업_인프라정비.txt

## Simondon Mapping
이접의 해소(resolution of disparation): AI가 만든 "메모앱" 프레이밍과 한빈의 "개인 온톨로지" 비전 사이의 긴장이, 원본 텍스트 기반 ANCHOR를 통해 새 차원에서 해소됨.
