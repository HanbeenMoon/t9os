# ADR-071: Remote Control Always-On

- Date: 2026-03-23
- Status: Accepted
- Decision: Add Claude Code `--rc` flag to bashrc alias, enabling remote control on every session automatically.
- Rationale: Mobile access to CLI sessions should be frictionless. Manual flag entry is a UX violation.
- Outcome: `.bashrc` alias with `--rc`

## Simondon Mapping
Expanding the associated milieu — mobile access extends the environment in which individuation can occur.
