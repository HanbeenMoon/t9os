# ADR-031: T9 OS v0.2 시몽동 전면 개정

- 날짜: 2026-03-16
- 상태: 채택됨
- 결정: T9 OS 전체를 시몽동 개체화 이론 6개 핵심 개념(disparation, transduction, associated milieu, concretization, modulation, preindividual)으로 통일한다. L1/L2/L3 헌법, t9_seed.py 엔진, CLAUDE.md를 동시에 v0.2로 개정한다.
- 이유: v0.1은 시몽동 개념이 산발적으로 적용되어 일관성 부족. 리서치 에이전트 7개 병렬 투입으로 시몽동 원전 133KB 분석 완료 후, 모든 계층에 통일 적용이 가능해짐.
- 대안: 점진적 개념 추가 (일관성 확보 어려움), 다른 철학 프레임워크 (이미 시몽동에 투자)
- 결과: SIMONDON.md 리서치 문서 생성, DB 스키마 보강(relates 테이블, tension 감지), 12개 상태 전이 그래프 확정, 다이제스트 343건 FTS 인덱싱
- 출처: 20260316_CC_003_041500_T9OS_v02_시몽동개정_FinBot_ODNAR.txt, commit 3c361095

## Simondon Mapping
모듈레이션(modulation): T9 OS 자체가 연속적으로 자기 형태를 재조형하는 과정. v0.1→v0.2는 시스템 수준의 개체화.
