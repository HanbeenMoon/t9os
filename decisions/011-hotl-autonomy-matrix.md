# ADR-011: HITL to HOTL Transition — Autonomy Tiers

- Date: 2026-03-19
- Status: Accepted
- Decision: Transition from Human-in-the-Loop (HITL) to Human-on-the-Loop (HOTL) with three autonomy tiers: T1 (fully autonomous), T2 (post-report), T3 (pre-approval).
- Rationale: HITL requires designer intervention for every decision, creating bottlenecks — especially during long/overnight sessions where the agent idles waiting for input.
- Outcome: Tier classification in L1, guardian-enforced boundaries

## Simondon Mapping
Concretization — as tools integrate more deeply, the human role shifts from micro-management to direction-setting, mirroring abstract-to-concrete evolution.
