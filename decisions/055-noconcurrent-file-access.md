# ADR-055: 같은 파일 동시 접근 금지 — 병렬 세션 규칙

- 날짜: 2026-02-21 (초기 규칙), 2026-03-16 (L1 명시)
- 상태: 채택됨
- 결정: 병렬 실행 중인 에이전트(cc, cx, gm)가 같은 파일에 동시 접근하는 것을 절대 금지한다. 긴 스크립트는 cx가 생성+실행 한번에 처리하고, 짧은/중간 스크립트는 cc 여러 인스턴스가 병렬 처리하되 파일 겹침을 확인한다.
- 이유: 동시 편집 시 내용 덮어쓰기, 충돌, 데이터 손실 발생. Git merge conflict보다 심각한 실시간 파일 손상 가능.
- 대안: 파일 잠금 메커니즘 (fcntl flock, 복잡성 증가), 낙관적 동시성 (손실 위험)
- 결과: CLAUDE.md 절대 규칙, claim/release(ADR-006)와 연계
- 출처: CLAUDE.md 섹션 4 절대 규칙, 20260221_CC_007_130351_claude_md_update.txt
